Animated Data Visualization

link to animated visualization

This data visualization includes the following elements:

  • A list of stars that have planetary systems that can be arranged by system size or discover date
  • The size of the circles representing the stars and planets are relative to their real life sizes
  • The size of the radius is relative to the real life radius of each planet
  • The speed of the orbit of each planet is relative to the real orbital speed
  • If the system is a binary star system, two stars are shown orbiting a central point at a relative speed
  • If you hover over one of the systems, there is a pop up of information that includes the systems name and how many planets there are
  • Depending of who much is known about the system, more information may pop up in the form of descriptions or images
If you scroll to the Kepler 62 system, you can see an example of extra information and images. Also, Kepler 47 shows extra information, images, and a binaray star system. Our solar system is shown at the bottom for comparison.

To re-create this visualization you can use the following techniques:

  • Event listeners are used on the buttons to organize the list of data, when clicked the visualization is re-drawn
  • Event listeners are also used for the hover elements for showing the system names and more information
  • You can use webkit to control the movement of the planets and stars, if in a binary system, in their orbits
  • The labels and extra information that shows up when a system is hovered over can be created by drawing a rectangle, using mouseon, and adding the information. You then remove it, using mouseout, after it is no longer being hovered over.
